— Rule
Arrow-right (→) whenever the button leads to another page on the site. Arrow-NE (↗) when leaving for an external URL. Arrow-down (↓) for downloads. Action buttons that stay on the current page (play, add to cart, toggle) use an action icon — never an arrow. Arrows slide 3px on hover.
